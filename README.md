# Face API

An app that uses face recognition to apply filters to a users face in real time via their webcam. The app detects facial expressions and applies a corresponding filter. Facial expressions used: neutral, happy, angry and surprised.

**Programming Language**: JavaScript

**Libraries**: [face-api.js](https://github.com/justadudewhohacks/face-api.js/)

**Markup/Styling**: HTML5:Canvas/CSS3

---

# Setup

- Download the files

- Install live-server:

```
npm install -g live-server
```
- Navigate to project folder and run:

```
live-server
```
